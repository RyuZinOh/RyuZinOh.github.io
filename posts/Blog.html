    Java applets are pretty old school now and not widely used anymore because modern web technology like html, css, js has taken over, but backen in those days , appplets in java were used as a key tool for creative a interactive web projects

    sample code

    click to view code import java.applet.Applet;
    import javat.awt.Graphics;

    public class HelloWordApplet extends Applet{
      public void paint(Graphics g){
        g.drawString("Hello world", 20, 20)
      }
    }

    What's the process ?

    1. java.applet.Applet is the base cass for creating applet, and the java.awt.Graphics is the class that provides context for drawing on the applet
    2. we created the class HelloWordApplet class which extends the Applet class
    3. paint method is used to drawing purposes
    4. Graphics class drawstring is used to draw string on cooridanate (20,20)

    in order to run this applet , we hvae to create html and link it thee using applet tag like this
    click to view code <applet code = "HelloWordApplet">
